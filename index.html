Java Script (Mocha/ Livescript):

  • JavaScript is a programming language created by Brendan Eich in 1995 used to create dynamic and interactive web pages.

  • JavaScript allows developers to implement complex features on web pages, such as dynamic content, form validation, animations, and more.

  • JavaScript is a single-threaded language that executes one task at a time.

  • It is an interpreted language which means it executes the code line by line.

  • used for frontend & backend boths.

  • In 1997, Netscape submitted JavaScript to ECMA International, aiming to create a standardized language that everyone can use. The standard rules was called ECMA-262 and the language “ECMAScript”

  • European Computer Manufacturers Association (ECMA): is a non-profit, global standards organization Founded in 1961. Its mission is to create rules that help different technologies work well together, so new products can work smoothly with each other. Many big tech companies are members, like Google, Microsoft, Apple, IBM, Meta, and others.

  • Below is a summary of the major ECMAScript versions:

    • ECMAScript 1 (1997): The first edition, introducing the core features and syntax of the language.

    • ECMAScript 2 (1998): it's improved the features of ECMAScript 1 and made the original specification clearer and better.

    • ECMAScript 3 (1999): Added regular expression, strict equality (===), try-catch.

    • ECMAScript 4 (Not released): A planned major update that was discarded due to stackholders disagreements.

    • ECMAScript 5 (2009): Added strict mode, JSON support, and getter/setter functions.

    • ECMAScript 6 (2015): Also known as ES6 or ES2015; introduced classes, modules, arrow functions, destructing, let and const, promises, Template literals, and much more.

    • ECMAScript 16 (2016): Continued to add features like async/await, optional chaining, Exponentiation operator (**) and more.

    • ECMAScript 17 (2017): Introduced features like SharedArrayBuffer, Atomics, and async iteration.

    • ECMAScript 18 (2018): Added features such as rest/spread properties, asynchronous iteration, and improvements in regular expressions.

    • ECMAScript 19 (2019): Included features like Array.flat(), Array.flatMap(), Object.fromEntries(), and optional catch binding.

    • ECMAScript 20 (2020): Introduced BigInt type, nullish coalescing (??), globalThis, optional chaining (?.).

    • ECMAScript 21 (2021): Added logical assignment operators, Numeric Separators, Promise.any(), WeakRefs, and more.

    • ECMAScript 22 (2022): Included features like Top-level await, class static initialization blocks, and symbolic properties.

    • ECMAScript 23 (2023): Continues to enhance the language with features like Symbol.prototype.description, Array.prototype.findLast and findLastIndex, and more syntax improvements.

    • ECMAScript 24 (2024): Introduced Object.groupBy, Map.groupBy, Promise.withResolvers, set operations on Set.prototype, Unicode flag for regular expressions

JavaScript operators:

are special symbols used to perform operations on variables and values. their have some type of operators.
  1. Arithmetic Operators: used to perform basic mathematical operations like addition, subtraction, multiplication, and divisions. 
                let sum = 10 + 5; // addition
            let difference = 10 - 5; // subtraction
            let product = 10 * 5; // multiplication
            let quotient = 10 / 5; // division
            let remainder = 10 % 3; // modulus
            let increment = ++count; // increment
            let decrement = --count; // decrement
            let exponentiation = x**2 // raises the first operand to the power of the second operand

  2. Assignment Operators: used to assign values to variables, such as =, +=, -=, etc. 
                let x = 10; // simple assignment
            x += 5; // adds 5 to x, so x becomes 15    //       x += 5 == x = x + 5
            x -= 3; // subtracts 3 from x, so x becomes 12 //   x -= 5 == x = x - 5
            x *= 4; //multiplied 4 with x, so x x becomes 48            
            x /= 2; // // divides x by 2, so x becomes 24
            x %= 5; // assigns the remainder of x divided by 5 to x

  3. Comparison Operators: used to compare two values, and return a boolean value. , e.g., ==, ===, !=, !==, >, <, >=, <=. Example: 
                let isEqual = (5 == '5'); // true, equal value
            let isStrictEqual = (5 === '5'); // false, different types
            let isGreater = (10 > 5); // true
            let isGreatorOrEqual = (5 >= 5); // true
            let isLessthen = (5 < 10); // true
            let isLessOrEqual = (5 <= 5); // true
            let notequal = (5 != 3); // true,

  4. Logical Operators: used to combine conditional statement and return a boolean result. such as && (and), || (or), ! (not). Example: 
                let andResult = (true && false); // false
            let orResult = (true || false); // true
            let notResult = !true; // false
            Logical AND (&&):	Returns true if both statements are true	(x > 1) && (y < 10).
            Logical OR (||):	Returns true if one statement is true	(x == 5) || (y == 5)
            Logical NOT (!):	Reverses the logical state of an operand	!(x == y)

  5. Ternary Operator (Conditional): a shorthand for if-else statement, written as condition ? expr1 : expr2. 
                let age = 20;
            let status = age >= 18 ? "Adult" : "Minor";
            console.log(status);  // Adult

  6. Nullish Coalescing Operator (??): It returns the right-hand operand when the left-hand operand is null or undefined; Example 
                let name = null;
            let user = name ?? "Guest"; // user is "Guest"
            let count = 0;
            let value = count ?? 1; // value is 0 (because 0 is not null/undefined)

CSS Animations:

through animations we can change the element style from one to another. thier have some properties and rules.
  1. @keyframes: is a function for animation used to control the animation properties in deferent points for an element. it,s have two values
    • From: which define the starting point of an element animation.
    • To: which define the ending point of an element animation
    • Note: this value we can give in % like the starting point have 0% and ending point have 20%, 30%, 100% or which we wants.
      • Example of @keyframes: 
                    @keyframes animation-name{
                from{
                  top: 30px;
                  left: 20px;
                }

                to{
                  top: 70px
                  left: 50px
                }
              }

              or we can use % values Like

              @keyframes animation-name{
                0%{
                  top: 10px;
                  left: 15px;
                }

                20%{
                  top: 20px;
                  left: 25px;
                }

                50%{
                  top: 60px;
                  left: 70px;
                }

                100%{
                  top: 90px;
                  left: 120px;
                }
              }
    it is the basic example of @keyframe function animation


  2. animation-name Property: specifies a name for an element to bind this element with @keyframe. like 'animation-name: animatNamepro;'


  3. animation-duration Property: it specifies how many time an animation should take to complete one cycle. like ' animation-duration: 4s;'


  4. animation-delay Property: it Specifies a delay for the start of an animation. like 'animation-delay: 3s';
    here it,s wait for 3 seconds then start animtaions.


  5. animation-iteration-count Property: it specifies how many times the animation should run. like 'animation-iteration-count: 5 / infinite;'
    'infinite' mean its iteration-count have Continuesly runing. like 'animation-iteration-count: infinite;'


  6. animation-direction Property: it specifies the animation should be played forwords, backwrods, or in alternate cycle. lik 'animation-direction: normal/ reverse/ alternate/ alternate-reverse;'
    • normal: the animation as played as normal (forward).
    • reverse: the animation is played in reverse direction (backward).
    • alternate: The animation is played forwards first, then backwards
    • alternate-reverse: The animation is played backwards first, then forwards


  7. animation-timing-function Property: it Specifies the speed curve of the animation. that have some values.
    • ease: Specifies an animation with a slow start, then fast then end slow.
    • linear: Specifies an animation with the same speed from start to end.
    • ease-in: Specifies an animation with a slow start
    • ease-out: Specifies an animation with a slow end
    • ease-in-out: Specifies an animation with a slow start and end
    • cubic-bezier(n,n,n,n): Lets you define your own values in a cubic-bezier function

JavaScript Storage:

it,s enables websites to store data locally within a user's browser as key-value pairs. it,s have some types like.
  1. Local Storage: Data stored in localStorage persists even after the browser is closed and reopened. The data does not expire unless explicitly deleted. this data is accessible across different pages of the same website. but not access across different websites or tabs. it,s capacity have 5MB to 10MB.


  2. Session Storage: Data stored in sessionStorage is only available for the duration of the page session. mean It stores data for the page until the tab or browser is closed. this data is accassable only within the same tab. it,s capacity have 5MB.


  3. Cookies: it can have an expiration date or be session-based. If no expiration is set, they are deleted when the browser is closed. Cookies are sent with every HTTP request(server side as well as client side). it,s capacity have 4KB.


    4. setItem() method: use to store data as a key value pairs in local storage.

    getItem() method: use to get data from local storage.

    removeItem() method: use to remove an item from local storage.

    clear() method: use to clearly remove all the data from local storage.
